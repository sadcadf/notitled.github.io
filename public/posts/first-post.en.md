# Welcome to my Blog!

## Why Markdown?

Markdown is a simple and effective way to write content. Here are some benefits:

- **Simplicity**: focused on content, not formatting
- **Portability**: files are easy to edit in any text editor
- **Readability**: source text is easy to read even without rendering
